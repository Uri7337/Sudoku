Inside sudoku-container we generate an 2D array using JS.

//Sudoku fields
let fields = new Array(9);
// variable to hold all the field blocks in a String
let finalGrid = "";
let sudokuGridContainer = document.getElementById("sudoku-container");

function addToFinalGrid(value) {
  finalGrid += "<div class=\"field\">" + value + "</div>";
}

//fill fields with ones
for (let i = 0; i < 9; i++) {
    fields[i] = new Array(9);
    fields[i].fill(1);
    fields[i].forEach(addToFinalGrid);
}

sudokuGridContainer.innerHTML = finalGrid;

Sudoku Generator

Then, we can generate the numbers in canvas.

We are going to use the "Naive approach" - as they call it on geeksforgeeks

Naive approach

  1. Randomly take any number 1-9.
  2. Check if it is safe to put in the cell. (row, column and box)
  3. If safe, place it and increment to the next location and go to step 1.
  4. If not safe, then without increment go to step 1.
  5. Once the matrix is filled, remove k elements randomly to complete the game.

Valid Sudoku

  • Each of the digits 1-9 must occur exactly once in each row.
  • Each of the digits 1-9 must occur exactly once in each column.
  • Each of the digits 1-9 must occur exactly once in each of the 9, 3x3 sub-boxes of the grid.

Dictionary

Chaining selectors(comma-separated): multiple selectors applied with the same style. Notice the , in:
div:nth-child(n+19):nth-child(-n+27),div:nth-child(n+46):nth-child(-n+54)

Composition: you can combine multiple pseudo-classes together (:nth-child(...):nth-child(...))
