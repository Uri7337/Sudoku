Game Engine

Goals for engine are:

  • Sudoku generator

  • User friendly page to solve sudoku puzzle

    • Sudoku Canvas

    First, we have to create canvas for sudoku.
    That is done by: 

    <div class="canvas">
...
</div>

    Then we have: 

    <div id="sudoku-container">
...
</div>

    There are also styles for the canvas and sudoku container.
    Styles for canvas are not much interesting, they are just sayng that canvas should be in the middle of the screen.
    For the sudoku-container on the other hand, we have sone interesting display grid: 

    #sudoku-container {
  display: grid;
  grid-template-columns: repeat(9, 1fr);
  padding: 10px;
  border-radius: 5px;
}

    And inside sudoku-container we have for every number div element with class field(more info in JS below). 

    #sudoku-container > div {
    background-color: #f1f1f1;
    border: 1px solid black;
    text-align: center;
    display: flex;
    justify-content: center;
    align-items: center;
}

    Inside sudoku-container we generate an 2D array using JS.

    //Sudoku fields
let fields = new Array(9);
// variable to hold all the field blocks in a String
let finalGrid = "";
let sudokuGridContainer = document.getElementById("sudoku-container");

function addToFinalGrid(value) {
  finalGrid += "<div class=\"field\">" + value + "</div>";
}

//fill fields with ones
for (let i = 0; i < 9; i++) {
    fields[i] = new Array(9);
    fields[i].fill(1);
    fields[i].forEach(addToFinalGrid);
}

sudokuGridContainer.innerHTML = finalGrid;
